## 📖 Sobre o Projeto

O **Axiom Calc** é uma calculadora desenvolvida em **C++**, executada via terminal (**CLI**), com foco em organização, modularização e boas práticas.

O projeto começou como um único arquivo `.cpp` e foi totalmente refatorado para uma estrutura mais limpa e profissional, separando responsabilidades sem alterar a lógica original.

---

## ✨ Funcionalidades

A calculadora conta com **24 operações matemáticas**, organizadas por nível:

### 🔢 Básicas
1. Soma  
2. Subtração  
3. Multiplicação  
4. Divisão  
5. Par ou ímpar  
6. Porcentagem  
7. Média aritmética  

---

### 📐 Intermediárias
8. Potência  
9. Raiz  
10. Média ponderada  
11. Módulo (valor absoluto)  

---

### 📊 Avançadas
12. Logaritmo  
13. Funções trigonométricas  
14. Fatorial  
15. Conversões  
16. Exponenciais especiais  
17. Arredondamento  
18. Máximo e mínimo  
19. Funções estatísticas  
20. Equações  
21. Sequências matemáticas  

---

### 🧪 Complexas
22. Regressão linear  
23. Integral simples (numérica)  
24. Derivada numérica

## 🚀 Como Executar

### 1. Compile o projeto:

```bash
g++ src/*.cpp src/operations/*.cpp -Iinclude -o main.exe
```

### 2. Execute:

```bash
.\main.exe
```
